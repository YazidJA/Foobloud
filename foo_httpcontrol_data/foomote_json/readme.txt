########################################################
#                                                      #
#               HOW TO INSTALL TEMPLATE:               #
#                                                      #
########################################################

# foomote_json template only works with FooMote version 3.0 and above 
# Copy the foomote_json folder of this zip file to
	# %APPDATA%/foobar2000/foo_httpcontrol_data if foobar2000 is installed in standard mode
	# <DIRECTORY_OF_FOOBAR2000>/foo_httpcontrol_data if foobar2000 is installed in portable mode
# The foo_httpcontrol_data directory should now contain the foomote_json folder of this zip file
